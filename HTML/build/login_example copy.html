What makes a login screen accessible?


Many of us log into websites and applications multiple times per day, and it's easy to take for granted what's under the hood. As I researched best practices for creating a11y-friendly login screens, I came up with a list that I adhered to when creating the login screen below.


Here's the list:


  • Create focus on text input fields
  • Use a label element for each form control
  • Add "For" attributes and make sure to match their corresponding id value
  • Use aria-required=true, to let screen readers communicate required steps
  • Use a color contrast checker, to make ensure readability and meet WCAG 2.0 standards
